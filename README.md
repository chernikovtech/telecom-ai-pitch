# Telecom × AI — Yango Tech Pitch Site

Strategic pitch microsite positioning telecoms as the next-generation AI distribution layer. Targets telecom operators across Africa, Indonesia, and MENA.

**Brand:** Yango Tech design system (from tech.yango.com)

## Local Dev

```bash
npm install
npm start
# → http://localhost:3000
```

## Deploy to Railway

1. Push to GitHub
2. Connect in Railway dashboard
3. Add custom domain: `telecom-ai.chernikov.tech`
4. Cloudflare CNAME: `telecom-ai` → Railway domain (Proxied)

- **/** — Strategic thesis (connectivity → AI distribution)

- **/b2b.html** — Enterprise AI products with mockups
- **/b2c.html** — Consumer AI products with phone mockups
